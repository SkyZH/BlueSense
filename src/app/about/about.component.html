Introduction

BlueSense is an open-source IoT platform for domestic environment monitoring. Data are collected from Arduino and Raspberry Pi in SkyZH's home.

Sensors and Devices

  • Raspberry Pi 3 Model B
  • Serial via BLE
  • DFRobot Bluno Mega 2560
  • Serial & I2C
  • Gravity: I2C BME280 Environmental Sensor (Temperature, Humidity, Barometer)
  • PM2.5 Laser Dust Sensor

Building of BlueSense

BlueSense is built with Firebase and Angular. Realtime data are refreshed every minute.

You can follow SkyZH on Twitter and GitHub.
